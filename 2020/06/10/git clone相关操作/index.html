git clone相关操作
发表于 2020-06-10| 更新于 2020-06-10|经验
字数总计:178|阅读时长: 1 分钟
| 阅读量:|评论数:

第一步：在github上找到心仪的仓库，点击“Clone or download”，复制url

第二步：在电脑上找好要放下载的东西的文件夹，在里面右击，然后选择“Git Bash Here”，就会弹出Git Bash（哦！所以每次不用cd了啊，直接在相关文件夹里右击就行了啊）

第三步：输入git clone https://github.com/……，然后就等着下载完吧

你以为这就完了吗？？？Naive！！！！！git clone一些比较大的项目时下载速度很慢诶！！所以这时可以借助国内镜像网站github.com.cnpmjs.org

用法就是替换上面第三步为git clone https://github.com.cnpmjs.org/……，然后就等下载完就可以了
